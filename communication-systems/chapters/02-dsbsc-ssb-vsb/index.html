DSB-SC (Double Sideband Suppressed Carrier)

By removing the carrier from AM, all power is in the sidebands:

Key Result — DSB-SC Signal
$$s_{DSB}(t) = A_c\, m(t)\cos(2\pi f_c t)$$
The carrier amplitude is directly proportional to m(t), with phase reversals at zero-crossings.
Derivation: Phase Reversals and Coherent Demodulation

When $m(t)$ changes sign, the factor $m(t)\cos(2\pi f_c t)$ undergoes a 180° phase shift. The spectrum is:

$$S_{DSB}(f)=\frac{A_c}{2}[M(f-f_c)+M(f+f_c)]$$

Bandwidth: $B_T = 2W$ (same as AM). Efficiency: 100% — all power carries information.

Coherent demodulation: Multiply by $\cos(2\pi f_c t + \phi)$ and LPF:

$$y(t)=\frac{A_c}{2}m(t)\cos\phi$$

A phase error $\phi$ scales the output by $\cos\phi$. At $\phi = 90°$, the output vanishes entirely — motivating the Costas loop for carrier recovery.

SSB (Single Sideband)

Key Result — SSB (USB) Signal
$$s_{USB}(t) = \frac{A_c}{2}\left[m(t)\cos(2\pi f_c t) - \hat{m}(t)\sin(2\pi f_c t)\right]$$
The Hilbert transform $\hat{m}(t)$ shifts all frequency components by 90°, enabling sideband cancellation.
Derivation: SSB via Analytic Signal

Define the analytic signal $m_+(t) = m(t) + j\hat{m}(t)$. Its spectrum is one-sided: $M_+(f)=2M(f)u(f)$.

The SSB signal (USB) is obtained by modulating $m_+(t)$:

$$s_{USB}(t)=\text{Re}\left\{\frac{A_c}{2}m_+(t)e^{j2\pi f_c t}\right\}$$

Expanding the real part yields the expression above. For LSB, replace $-\hat{m}(t)$ with $+\hat{m}(t)$.

Bandwidth: $B_T = W$ — half that of DSB-SC. Power efficiency: 100%. SNR figure of merit: 1 (same as DSB-SC).

VSB (Vestigial Sideband)

Derivation: VSB Filter Symmetry Condition

After filtering DSB-SC with $H_i(f)$ and coherent demodulation, the output spectrum is:

$$Y(f) \propto M(f)[H_i(f-f_c)+H_i(f+f_c)]$$

For distortion-free output: $H_i(f-f_c)+H_i(f+f_c)=\text{const}$ for $|f| \le W$.

This is a symmetry condition about $f_c$: the filter's transition band must have odd symmetry around the carrier.

The VSB bandwidth is $B_T \approx W + f_v$ where $f_v$ is the vestigial bandwidth retained from the unwanted sideband.

Example: NTSC TV uses $f_v = 0.75$ MHz, $W = 4.2$ MHz, so $B_T = 4.95$ MHz.
