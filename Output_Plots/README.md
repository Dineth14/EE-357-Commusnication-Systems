# Simulation Output Plots

This directory contains the graphical outputs generated by the simulation scripts in this repository.

## Files

### [natural_sampling_plots.png](natural_sampling_plots.png)
- **Description**: Visualizes the process of natural sampling and demodulation.
- **Contents**:
  - **Time Domain**: Original signal, Switching Signal (Pulse Train), Sampled Signal, Demodulated Signal.
  - **Frequency Domain**: Original Spectrum, Sampled Spectrum (showing harmonics), Demodulated Spectrum (showing baseband recovery).
- **Source Script**: [Natural_sampling.py](../Sampling/Natural_sampling.py)
- **Parameters**: $B=5$ Hz, $f_s=20$ Hz, $d=1/3$.

### [Square_Law_Modulation_Output.png](Square_Law_Modulation_Output.png)
- **Description**: Visualizes Amplitude Modulation using Square Law Modulation.
- **Contents**:
  - Unmodulated Carrier and Message Signals.
  - Modulated Signal in Time Domain.
  - Frequency Spectrum showing Carrier and Sidebands.
- **Source Script**: [Square_Law_Modulation.py](../Modulation/Square_Law_Modulation.py)

### [Flat_Top_Sampling_Output.png](Flat_Top_Sampling_Output.png)
- **Description**: Visualizes Flat Top Sampling (Sample and Hold) and the Aperture Effect.
- **Contents**:
  - **Time Domain**: Original vs held pulses.
  - **Demodulation**: Recovered signal using LPF.
  - **Frequency Domain**: Spectrum showing the "sinc" roll-off (Aperture Effect) affecting the harmonics.
- **Source Script**: [Flat_Top_Sampling.py](../Sampling/Flat_Top_Sampling.py)

### [PWM_Output.png](PWM_Output.png)
- **Description**: Visualizes Pulse Width Modulation (Trailing-Edge).
- **Contents**:
  - **Modulation**: Pulses with variable width $\tau_n = \tau_0 + k_w m(nT_s)$.
  - **Demodulation**: Recovered signal via Low Pass Filtering.
- **Source Script**: [PWM.py](../Modulation/PWM.py)

### [PPM_Output.png](PPM_Output.png)
- **Description**: Visualizes Pulse Position Modulation.
- **Contents**:
  - **Modulation**: Fixed-width pulses shifted from clock ticks by $k_p m(nT_s)$.
  - **Conversion**: Intermediate conversion to PWM for recovery.
  - **Demodulation**: Signal recovered from converted PWM.
- **Source Script**: [PPM.py](../Modulation/PPM.py)
