Pulse Modulation

PAM, PWM (PDM), PPM — analog pulse modulation techniques that encode information in pulse amplitude, width, or position.

Overview

Pulse modulation techniques represent analog information using discrete pulses. Unlike PCM (Chapter 6), these are still analog-to-analog representations — the information resides in a continuous parameter of the pulse train.

  • PAM — Pulse Amplitude Modulation: information in pulse heights
  • PWM/PDM — Pulse Width/Duration Modulation: information in pulse widths
  • PPM — Pulse Position Modulation: information in pulse positions

Pulse Amplitude Modulation (PAM)

PAM is the simplest form — the amplitude of each pulse is proportional to the message signal at the sampling instant:

$$s_{PAM}(t)=\sum_{n}m(nT_s)\,p(t-nT_s)$$

Two forms exist:

  • Natural PAM: Pulse tops follow the signal shape (natural sampling)
  • Flat-top PAM: Pulse tops are flat at the sampled value (sample-and-hold)

PAM is the basis for all other pulse modulation schemes and is the first step in PCM encoding.

PAM spectrum analysis

For flat-top PAM with pulse width $\tau$:

$$S_{PAM}(f)=\tau\,\text{sinc}(f\tau)\cdot f_s\sum_n M(f-nf_s)$$

The sinc envelope causes aperture distortion that increases with $\tau$. Equalization filter: $H_{eq}(f)=\frac{1}{\text{sinc}(f\tau)}$ for $|f|\le f_m$.

Pulse Width Modulation (PWM / PDM)

In PWM, the width (duration) of each pulse is proportional to the message amplitude:

$$\tau_n = \tau_0\bigl[1 + k_w\,m(nT_s)\bigr]$$

where $\tau_0$ is the unmodulated pulse width and $k_w$ is the modulation sensitivity. Advantages:

  • Constant amplitude → easy to regenerate (noise immune pulse shape)
  • Information in pulse edges → simple demodulation by integration

Demodulation: integrate each pulse period. The integral is proportional to the pulse width, hence to $m(nT_s)$.

Bandwidth: PWM requires higher bandwidth than PAM due to the fast rise/fall edges needed to accurately encode width information.

Pulse Position Modulation (PPM)

In PPM, the position (time shift) of each pulse relative to a reference is proportional to the message:

$$\delta_n = k_p\,m(nT_s)$$

where $\delta_n$ is the time shift of the $n$-th pulse. PPM can be derived from PWM by differentiating the PWM signal and keeping only the trailing edges.

PPM advantages over PWM:

  • Higher power efficiency (constant, narrow pulses)
  • Better noise performance (timing jitter easier to combat than width distortion)

PPM disadvantage: requires synchronization — the receiver must know the reference positions.
