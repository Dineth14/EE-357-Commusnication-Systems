The Sampling Theorem

The Nyquist-Shannon sampling theorem states that a band-limited signal with maximum frequency $f_m$ can be perfectly reconstructed from its samples if the sampling rate satisfies:

$$f_s \geq 2f_m$$

The minimum sampling rate $f_s = 2f_m$ is called the Nyquist rate, and $f_N = f_s/2$ is the Nyquist frequency (the maximum frequency that can be represented).

Ideal (Impulse) Sampling

Mathematically, ideal sampling multiplies the signal by a Dirac comb:

$$m_s(t) = m(t) \cdot \delta_T(t) = m(t)\sum_{n=-\infty}^{\infty}\delta(t - nT_s)$$

In the frequency domain, this produces periodic replicas of $M(f)$:

$$M_s(f) = f_s \sum_{n=-\infty}^{\infty} M(f - nf_s)$$
Derivation: Frequency domain sampling

The Fourier transform of the Dirac comb is another Dirac comb:

$$\mathcal{F}\{\delta_T(t)\} = f_s\sum_{n=-\infty}^{\infty}\delta(f-nf_s)$$

Multiplication in time → convolution in frequency:

$$M_s(f) = M(f) * f_s\sum_n\delta(f-nf_s) = f_s\sum_n M(f-nf_s)$$

If $f_s\ge 2f_m$, the replicas don't overlap and $M(f)$ can be recovered with an ideal LPF of bandwidth $f_s/2$.

Natural Sampling

In natural sampling, a periodic pulse train (duty cycle $d = \tau/T_s$) gates the signal. The tops of the pulses follow the signal shape:

$$m_{ns}(t) = m(t) \cdot p(t), \quad p(t) = \sum_n \text{rect}\!\left(\frac{t-nT_s}{\tau}\right)$$

The spectrum contains weighted replicas: $M_{ns}(f) = \sum_n c_n\,M(f-nf_s)$ where $c_n = d\,\text{sinc}(nd)$.

Flat-Top Sampling (Sample-and-Hold)

A sample-and-hold circuit holds each sample constant for the pulse duration $\tau$. This is equivalent to ideal sampling followed by convolution with a rectangular pulse:

$$m_{ft}(t) = \sum_n m(nT_s)\,\text{rect}\!\left(\frac{t-nT_s}{\tau}\right)$$

The spectrum is:

$$M_{ft}(f) = \tau\,\text{sinc}(f\tau)\cdot f_s\sum_n M(f-nf_s)$$

The $\text{sinc}(f\tau)$ factor causes aperture distortion, which can be compensated with an equalizer.

Aliasing

When $f_s < 2f_m$, spectral replicas overlap, causing aliasing — irreversible distortion. Aliased frequency components appear as false low-frequency content. Prevention:

  • Anti-aliasing filter: Low-pass filter before sampling to limit bandwidth to $f_s/2$.
  • Oversampling: Use $f_s \gg 2f_m$ to create a guard band.

Uniform Quantization

Quantization maps continuous amplitude values to a finite set of $L = 2^B$ levels, where $B$ is the number of bits:

$$\Delta = \frac{x_{max} - x_{min}}{L} = \frac{2V_p}{2^B}$$

The maximum quantization error is $\pm\Delta/2$. Two types:

  • Mid-rise: No level at zero; levels at $\pm\Delta/2, \pm 3\Delta/2, \ldots$
  • Mid-tread: A level at zero; levels at $0, \pm\Delta, \pm 2\Delta, \ldots$

Signal-to-Quantization-Noise Ratio (SQNR)

For a full-scale sinusoidal input with uniform quantization:

$$\text{SQNR (dB)} = 6.02B + 1.76$$
Derivation: SQNR formula

Quantization error $e$ is uniformly distributed in $[-\Delta/2, +\Delta/2]$. Noise power:

$$\sigma_e^2 = \frac{\Delta^2}{12}$$

For a sinusoid of peak amplitude $V_p = L\Delta/2$, signal power = $V_p^2/2 = L^2\Delta^2/8$:

$$\text{SQNR} = \frac{L^2\Delta^2/8}{\Delta^2/12} = \frac{3L^2}{2} = \frac{3\cdot 2^{2B}}{2}$$ $$\text{SQNR (dB)} = 10\log_{10}\!\left(\frac{3}{2}\cdot 2^{2B}\right) = 10\log_{10}\frac{3}{2} + 20B\log_{10}2 = 1.76 + 6.02B$$

Each additional bit improves SQNR by approximately 6 dB.

Signal Reconstruction

Perfect reconstruction uses the ideal interpolation formula (Whittaker-Shannon):

$$m(t) = \sum_{n=-\infty}^{\infty} m(nT_s)\,\text{sinc}\!\left(\frac{t - nT_s}{T_s}\right)$$

Practical reconstruction methods:

  • Zero-Order Hold (ZOH): Hold each sample constant — simple but introduces sinc distortion.
  • First-Order Hold (FOH): Linear interpolation between samples — better approximation.
  • Ideal LPF: Perfect reconstruction using sinc interpolation (non-causal, impractical).
